## 1. 검수 방법

각 행의 입력과 제안 결과를 확인하고 다음 중 하나로 판정했다.

- `승인`: 제안 결과를 웹 공식 기대값으로 사용
- `수정`: 올바른 기대값과 근거를 기입
- `보류`: 제작 규칙이나 추가 표본이 필요

사용자는 모든 업무·계산 검수 역할을 직접 담당한다. 아래 20건과 세 가지 계산 정책은 2026-07-19 사용자 승인으로 웹 공식 기준선이 됐다.

공통 입력:

- 두께: `2 mm`
- 내부 반경: `2 mm`
- 기준 cut angle: `135°`
- FIX 연신: V `1.2`, A `0.8`, NO-CUT `2`
- RATIO 컷 깊이: V `0.5`, A `0.8`, NO-CUT `0`
- 제품 길이: `2,400 mm`
- 수량: `10`

개별 행에 다른 조건이 있으면 해당 조건을 우선한다.

## 3. 반드시 결정할 쟁점

### 3.1 FIX 소수 연신 보존

레거시 MFC식 정수 변환은 `0.8` 또는 `0.4`처럼 절댓값이 1보다 작은 보정값을 0으로 만든다.

영향 표본:

- `WR-004`
- `WR-005`
- `WR-007`

승인 기준은 **FIX 소수 연신 보존**이다. 자동 보정의 소수값은 계산 중간값에 유지하고, 최종 구간 길이는 표본에 지정된 출력 자릿수와 처리 방식으로만 반올림한다. 이에 따라 `WR-004`는 `148.4 mm`, `WR-005`는 `151.6 mm`, `WR-007`은 `149.2 mm`로 승인됐다.

### 3.2 구간 계산 제외의 영향 범위

승인 기준은 **계산 제외 절곡을 절곡 양쪽 구간 모두에서 제외**하는 것이다. `WR-015`의 전개 폭은 `150 mm`로 승인됐다. 후속 모델·UI에서는 현재 `calculateElongation` 이름을 절곡 단위 의미가 분명한 이름으로 변경하는 작업을 계획한다.

### 3.3 cut angle 경계

승인 기준은 RATIO 계산을 `angle < cutAngle`일 때만 적용하는 것이다.

- `134°`: 적용
- `135°`: 미적용
- `136°`: 미적용

`WR-011`~`WR-013`은 이 경계로 승인됐다.

## 4. 아직 v1 계산 fixture에 포함하지 않은 항목

다음 기능은 결과를 추측하지 않고 P1의 테스트 우선 구현 항목으로 유지한다.

- 변수·수식 평가와 순환 참조

- `EL_OPTION_DG` 대각선 적용
- `EL_OPTION_EXT1` 확장 계산
- 곡선·호 길이·곡 깊이
- 박스의 실제 두 번째 전개 길이와 다중 DXF
