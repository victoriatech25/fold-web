## 데이터 해석

하나의 `FoldSegment`는 실제 길이를 가진 선이며 `bendAfter`는 그 선 끝의 절곡이다. 현재 선 계산에는 이전 선 끝의 절곡과 현재 선 끝의 절곡이 함께 영향을 준다.

## 고정(FIX) 방식

앞각의 기여값은 `+연신율`, 뒷각은 `-연신율`이다. 현재 구간의 보정값은 양 끝 기여값의 합이며 계산 길이는 다음과 같다.

```text
자동 보정 = 소수 보존(이전 절곡 기여 + 다음 절곡 기여)
계산 길이 = 출력 자릿수 규칙(입력 길이 - 적용 보정)
```

MFC는 합산 연신율을 소수 5자리로 반올림한 뒤 C++ `int`로 변환해 절댓값 1 미만을 0으로 만든다. 이 동작은 `LEGACY_DEFECT`로 판정했다. 웹 공식 기준은 FIX 자동 보정의 소수를 보존하며, 최종 구간 길이에만 설정된 소수 자리와 반올림·버림·올림 규칙을 적용한다.

`V-CUT`이 꺼지면 각 방향은 유지하지만 재질의 노컷 연신율을 사용한다.

`calculateElongation=false`인 선 끝의 절곡은 계산 대상에서 제외한다. 해당 절곡 기여는 절곡 앞 구간과 뒤 구간 모두에 적용하지 않는다. 후속 데이터 모델에서는 이 속성을 절곡 단위 의미가 분명한 이름으로 변경한다.

## 비율(RATIO) 방식

비율 방식은 연신율 대신 두께와 컷 깊이를 사용한다.

```text
앞각 기여 = 두께 - 컷 깊이
뒷각 기여 = -컷 깊이
계산 길이 = 입력 길이 - 이전 기여 - 다음 기여
```

절곡 각도가 재질의 `cutAngle`보다 작은 경우에만 적용한다. 경계값과 같거나 큰 각도에는 적용하지 않는다. `WEB-REFERENCE-V1`의 134°·135°·136° 경계 표본으로 승인됐다.

## 소수 처리

각 구간에 연신율을 적용한 후 설정에 따라 반올림, 버림 또는 올림한다. MFC의 `Gn_MathRound`는 두 번째 인자 1이 정수, 2가 소수 첫째 자리라는 특이한 규약을 사용하고 음수의 절반값은 0에서 멀어지는 방향으로 처리한다.

## 제품 크기

```text
전개 폭 = 모든 구간 계산 길이의 합
제품 크기 = 전개 폭(mm) × 제품 길이(mm)
개당 면적(m²) = 전개 폭 × 제품 길이 ÷ 1,000,000
총면적(m²) = 개당 면적 × 수량
```
