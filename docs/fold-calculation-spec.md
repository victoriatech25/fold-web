# 절곡 전개 계산 명세 (프로토타입 1단계)

## 목적

MFC `도면Pro`의 절곡 폭 계산을 UI와 분리된 TypeScript 엔진으로 재현한다. 이 문서는 코드에서 확인된 동작만 확정 규칙으로 기록한다.

## MFC 근거

- 각 타입: `_define.h`의 `Henum_AngleType`
- 재질별 값 선택: `DrawFx.cpp`의 `DFx_CutDepth_SetAt`
- 고정 계산: `_common.cpp`의 `Gn_Elongation_GetFixNA`
- 비율 계산: `_common.cpp`의 `Gn_Elongation_GetRatioOCW`
- 계산 흐름과 소수 처리: `Work03Dlg.cpp`의 `Fn_CalcLenW_SellFold`

## 데이터 해석

하나의 `FoldSegment`는 실제 길이를 가진 선이며 `bendAfter`는 그 선 끝의 절곡이다. 현재 선 계산에는 이전 선 끝의 절곡과 현재 선 끝의 절곡이 함께 영향을 준다.

## 고정(FIX) 방식

앞각의 기여값은 `+연신율`, 뒷각은 `-연신율`이다. 현재 구간의 보정값은 양 끝 기여값의 합이며 계산 길이는 다음과 같다.

```text
계산 길이 = 입력 길이 - 정수화(이전 절곡 기여 + 다음 절곡 기여)
```

MFC는 합산 연신율을 소수 5자리로 반올림한 뒤 C++ `int`로 변환한다. 결과적으로 절댓값 1 미만은 0이 되고, 1 이상은 0 방향으로 버려진다.

`V-CUT`이 꺼지면 각 방향은 유지하지만 재질의 노컷 연신율을 사용한다.

## 비율(RATIO) 방식

비율 방식은 연신율 대신 두께와 컷 깊이를 사용한다.

```text
앞각 기여 = 두께 - 컷 깊이
뒷각 기여 = -컷 깊이
계산 길이 = 입력 길이 - 이전 기여 - 다음 기여
```

절곡 각도가 재질의 `cutAngle`보다 작은 경우에만 적용한다. 경계값과 같거나 큰 각도에는 적용하지 않는다.

## 소수 처리

각 구간에 연신율을 적용한 후 설정에 따라 반올림, 버림 또는 올림한다. MFC의 `Gn_MathRound`는 두 번째 인자 1이 정수, 2가 소수 첫째 자리라는 특이한 규약을 사용하고 음수의 절반값은 0에서 멀어지는 방향으로 처리한다.

## 제품 크기

```text
전개 폭 = 모든 구간 계산 길이의 합
제품 크기 = 전개 폭(mm) × 제품 길이(mm)
개당 면적(m²) = 전개 폭 × 제품 길이 ÷ 1,000,000
총면적(m²) = 개당 면적 × 수량
```

## 검증 데이터 확보 기준

실제 MFC에서 최소 20개 도면에 대해 재질, 두께, V컷 여부, 각 선의 입력 길이·각 타입·각도·연신율·컷 깊이·계산 길이와 최종 전개 폭을 기록한다. 웹 결과는 구간별 계산 길이와 최종 폭 모두 일치해야 한다.
